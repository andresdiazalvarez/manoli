# Proyecto Manoli

Aplicación web instalable para controlar el estado de las viviendas de los edificios 1 y 2.

## Publicar en GitHub Pages

1. Crear un repositorio público en GitHub.
2. Subir a la raíz del repositorio todos los archivos y la carpeta `icons`.
3. Abrir **Settings > Pages**.
4. En **Build and deployment**, seleccionar **Deploy from a branch**.
5. Elegir la rama **main**, la carpeta **/(root)** y pulsar **Save**.

Cuando GitHub muestre la dirección publicada, se puede abrir desde el móvil e instalar desde el botón **Instalar aplicación**.

Los datos se guardan localmente en el dispositivo utilizado.
